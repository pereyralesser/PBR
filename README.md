# Panorama Bíblico Redentor — De la Creación a la Consumación

Sitio web estático (HTML + CSS) listo para publicar en GitHub Pages.

## Publicación en GitHub Pages

1. Creá un repositorio en GitHub, por ejemplo `panorama-biblico-redentor`
2. Subí todo el contenido de esta carpeta al repositorio (misma raíz)
3. En GitHub, entrá a Settings → Pages
4. En Build and deployment elegí
   - Source: Deploy from a branch
   - Branch: main, folder: / (root)
5. Guardá y esperá a que GitHub publique el sitio

## Estructura

- `index.html` portada
- `alumno/` lecciones con espacios en blanco
- `maestro/` lecciones con respuestas y notas
- `assets/` estilos e imágenes

No se incluye buscador y no hay dependencias de servidor.
